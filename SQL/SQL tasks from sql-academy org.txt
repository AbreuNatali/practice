Задание 6
Какие компании совершали перелеты на Boeing. Поля в результирующей таблице:
name

Выполнение задания. Запрос:
SELECT DISTINCT Company.name AS name FROM Trip JOIN Company
ON Trip.plane='Boeing' WHERE Trip.company=Company.id;
